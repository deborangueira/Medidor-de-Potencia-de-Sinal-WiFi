# Medidor de Potencia de Sinal WiFi

Nessa atividade, o intuito foi coletar dados referentes ao sinal do Wifi em dBm e publica-los em um dashboard que, sendo atualizado em tempo real, nós dá informações precisas sobre a potência do sinal. Para isso, o projeto conta com um **microcontrolador ESP32** conectado a um **Wifi**, medindo o sinal de radiofrequência por meio de uma **antena omnidirecional**, imprimindo os dados no monitor serial da **Arduino IDE** além de publicar esses dados num dashboard do **Ubdots**.
